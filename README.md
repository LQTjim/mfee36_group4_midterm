# 個人使用相關說明

請在`./parts` 下新增一個 `db-connect-local-config.php`
裡面內容大概如下:

```
<?php
$db_host='localhost'; # 有port號的範例: localhost:8889
$db_name='mfee_36_group_4';
$db_user='root';
$db_pass='root';
?>
```

目的是將資料庫的參數可以個人化。
檔名已經被 git ignore 排除 所以理論上是不影響本地端操作。
